# June 2026 AI Game Voting System

AIゲーム月間イベント「June 2026」の投票ページです。

## 🎮 投票ページ

👉 **[投票ページはこちら](https://ryoh0410-hub.github.io/bank-form/)**

## 📋 投票ルール

- 🗳️ **1端末につき1票のみ** - 投票後の変更はできません
- 📅 **開催期間**: 2026年6月1日（月）〜 6月21日（日）
- ⏰ **投票期限**: 2026年6月30日（火）まで
- 📣 **結果発表**: 2026年7月5日（日）以内に発表予定

## ✨ 機能

### ユーザー向け
- 10個のAIゲームから投票できます
- 投票後は他の作品の投票割合（%）が表示されます
- ブラウザのlocalStorageで投票記録を保存

### 管理者向け
- 右下の🔐ボタンをクリック
- パスワードを入力（パスワード: `Aigametubotubo`）
- ゲーム別の投票実数値とパーセンテージを確認可能

## 🛠️ 技術仕様

- **フロントエンド**: HTML5 + CSS3 + Vanilla JavaScript
- **ストレージ**: ブラウザのlocalStorage（外部API不要）
- **デプロイ**: GitHub Pages
- **ブラウザ互換性**: 全モダンブラウザに対応

## 📁 ファイル構成

```
bank-form/
├── index.html           # 投票ページ（トップページ）
├── voting_form.html     # 投票ページ（同じ内容）
├── README.md            # このファイル
└── bank_registration_form.html  # 別の登録フォーム
```

## 🔒 プライバシー

- 投票データはブラウザのlocalStorageのみに保存
- サーバーへの送信やトラッキングはありません
- 個人情報は収集しません
- 管理者はゲーム別の投票数のみ確認できます
